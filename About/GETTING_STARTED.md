# 📋 Before You Begin

## Requirements

TechPixel runs entirely in your browser.

You only need:

- 🌐 A modern web browser
  - Google Chrome
  - Microsoft Edge
  - Firefox
  - Safari

- 🌍 Internet access

That's it.

No downloads.
No installations.
No setup required.

---

# 🌐 Step 1: Open TechPixel

Navigate to:

> **https://techpixel.base44.app**

You'll immediately be taken into the TechPixel workspace.

If you simply want to explore, you can begin instantly using Guest Mode.

---

# 👤 Step 2: Choose How You Want to Use TechPixel

## Guest Sessions

- Quick questions

Benefits:

- Instant access
- No account required
- Low-friction onboarding

## Signed-In Sessions

Recommended for regular users.

Benefits include:

- Persistent chat history

- Access across sessions

# 📁 Step 4: Use WorldCode Templates

TechPixel includes built-in templates to help you get started faster.

Navigate to the template workspace and choose a starting point.

## Available Templates

### `killstreak.js`

Create systems that reward players for consecutive eliminations.

Use Cases:

- PvP arenas
- Minigames
- Ranked modes

---

### `spawn_entity.js`

Spawn entities dynamically using player coordinates.

Use Cases:

- Boss encounters
- Events
- NPC systems

---

### `team_balance.js`

Automatically balance teams.

Use Cases:

- Competitive matches
- Team-based minigames
- Fair matchmaking

---

### `shop_gui.js`

Build interactive shop interfaces.

Use Cases:

- Economy systems
- Currency mechanics
- Upgrade menus

---

### `api_lookup.ts`

Reference Bloxd APIs instantly.

Use Cases:

- Learning WorldCode
- Finding functions
- Understanding syntax

---

### `leaderboard.js`

Track competitive performance.

Use Cases:

- Seasonal rankings
- Kill leaderboards
- Score systems

# 🖼 Step 7: Generate Images with PixelGen

PixelGen is integrated directly into TechPixel.

Use it whenever visual inspiration is needed.

## Example Prompts

> Generate a Bloxd server logo.

> Create concept art for a medieval spawn.

> Design promotional artwork for my event.

> Visualize a fantasy boss arena.

---

## PixelGen Tips

- Be specific.
- Describe the style you want.
- Include themes and colors.
- Mention the intended purpose.

The more detail you provide, the better the results.

---

# 🧠 Step 8: Teach TechPixel with Memory

TechPixel can remember information across sessions.

Examples include:

> Remember that I prefer concise explanations.

> Remember that I'm building a prison server.

> Remember that I use WorldCode frequently.

---

## Managing Memory

You remain in control.

You can ask TechPixel to:

> Forget everything you know about me.

or

> Delete that memory.

at any time.
