# TechPixel Chat App - Future Roadmap

### 1. Custom System Prompt Editor
- [ ] **Dynamic Personality Toggling:** Allow the agent's core instructions, tone, behavior, and constraints to be rewritten without needing to redeploy or change the application's main code.
- [ ] **Variable Injection:** Support adding dynamic variables into the system prompt (e.g., current time, user profile data) so the agent stays highly context-aware during chats.
- [ ] **Branching Conversations:** Allow users to branch off from any previous message in the chat history to explore different response paths without losing the main thread.
- [ ] **Dictation / Hands-Free Input:** Integrate a real-time speech-to-text API to allow voice typing directly in the chat interface.
- [ ] **Smart Formatting & Punctuation**: Automatically clean up dictated text by adding proper punctuation, fixing filler words ("um," "uh"), and formatting paragraphs for cleaner AI processing.
- [ ] **Voice Commands (Hands-Free Control)**: Implement specific wake words or voice shortcuts to trigger actions, such as "Send message."
- [ ] **Multi-Agent Orchestration:** Implement a coordinator agent that can delegate complex tasks to specialized sub-agents (e.g., separate agents for coding, debugging, and documentation) and merge their outputs.
- [ ] **Inter-Agent Communication Bus:** Build a local message protocol allowing multiple agents to talk to each other, share state, and critique each other's work before presenting the final answer to the user.
- [ ] **Asynchronous Agent Swarms:** Allow background agents to run long-term research or code generation tasks concurrently without freezing the main chat UI window.
