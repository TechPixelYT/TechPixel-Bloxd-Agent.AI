## 🛠️ Want to Help?

If you are passionate about Bloxd.io, world-building, or scripting and want to help make TechPixel Agent even better, we would love your support! Here is how you can get involved:

* 🧪 **Help Test It:** Put the agent through its paces! Build things in your worlds using its code and let us know if you encounter any bugs, weird response layouts, or errors.
* 🧠 **Help Train It:** Submit your working World Code, Code Block scripts, and updated documentation through the **Admin Training Dashboard** so the agent can learn more.
* 💡 **Suggestions for Future Updates:** Have an awesome idea for a feature or UI upgrade? Share your thoughts so we can add them to our development roadmap.
* 📢 **Help Advertise It:** Share the chat link with your friends, your world members, and fellow builders in the Bloxd community to help grow our user base!
