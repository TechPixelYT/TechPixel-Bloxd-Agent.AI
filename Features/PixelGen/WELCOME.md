## 🚀 Core Philosophy
PixelGen isn't just a wrapper; it's a **context-aware orchestration layer**. Most AI generators fail at specific gaming aesthetics like voxels or low-poly pixel art. PixelGen solves this by using a proprietary multi-stage pipeline that "understands" the Bloxd.io visual language before the first pixel is even rendered.

***

## ✨ Key Features

### 🧠 Intelligent Prompt Enhancement (IPE)
Stop worrying about "masterpiece, 8k, highly detailed" keyword stuffing. 
* **Voxel Logic:** Automatically injects technical constraints for lighting, depth, and block-based geometry.
* **Semantic Expansion:** Converts a simple prompt like *"fire sword"* into a rich technical description involving elemental particle effects and stylized obsidian hilts.

### 🎮 Context-Aware Generation Modes
Specialized workflows tailored to the Bloxd.io development lifecycle:
* 👾 **Mob & Boss Concepts:** Generate multi-angle reference sheets for custom entities.
* 🏰 **World-Building:** Atmospheric generation for dungeons, sky-islands, and spawn points.
* ⚔️ **Item & UI Kit:** 1:1 ratio icons for inventory management and HUD elements.

### 🖼️ The Style Engine
Switch between multiple artistic directions instantly:
* **Native Voxel:** High-fidelity block art that fits the vanilla Bloxd aesthetic.
* **Cinematic:** High-contrast, moody renders for promotional banners or thumbnails.
* **Legacy Pixel Art:** Traditional 2D sprites for retro-style UI or texture packs.

## 🛠 Tech Stack & Architecture

### Frontend Excellence
* **Framework:** React + Vite for lightning-fast HMR and state management.
* **Styling:** Tailwind CSS using a **"Terminal-Dark"** theme for a professional developer aesthetic.
* **UX/UI:** Framer Motion powering sequenced grid-dissolve entrance animations and particle-based loading states.

### Backend Orchestration
| Component | Function |
| :--- | :--- |
| **InvokeLLM** | Handles the heavy lifting of prompt engineering and semantic expansion. |
| **GenerateImage** | The core rendering engine optimized for low-latency output. |
| **Entity Model** | Structured JSON-schema (`GeneratedImage`) for rigorous data integrity. |

## 💡 Workflow: From Concept to Asset

1. **Ideation:** Input your raw concept into the high-tech terminal prompt.
2. **Specialization:** Select your `Generation Type` (Weapon, Mob, Environment).
3. **Synthesis:** The AI runs a pre-check to enhance the prompt for the selected `Visual Style`.
4. **Validation:** Review the asset in the **Generation Gallery**.
5. **Deployment:** Bulk export assets with full metadata (prompts/seeds) for your project documentation.

## 📝 Metadata & Searchability
Every asset generated by PixelGen is a **Structured Entity**. 
* **Persistence:** All metadata (Styles, Types, Timestamps) is baked into the local database.
* **Discovery:** Advanced filtering allows you to search through your history by prompt keywords or generation parameters.
