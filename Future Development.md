# TechPixel Chat App - Future Roadmap

## High Priority Features

### 1. Custom System Prompt Editor
- [ ] **Admin Dashboard / Config Interface:** Build a secure settings panel or backend configuration file where the system prompt can be edited dynamically.
- [ ] **Dynamic Personality Toggling:** Allow the agent's core instructions, tone, behavior, and constraints to be rewritten without needing to redeploy or change the application's main code.
- [ ] **Variable Injection:** Support adding dynamic variables into the system prompt (e.g., current time, user profile data) so the agent stays highly context-aware during chats.
- [ ] **Prompt Versioning:** Implement a system to save and roll back to previous prompt versions if a new prompt tweak causes unexpected behavior.
